## Visual Design Notes

### Overall Design Language
The website uses a **minimalist, clean, and professional** design language with a focus on readability and whitespace.

### Color Palette
| Element | Color |
|---|---|
| Primary Background | Soft light blue/green gradient (`#EAF5FB` area) |
| Content Area | White (`#FFFFFF`) |
| Logo Badge | Teal/green with white "OO" text |
| Navigation Links | Blue (active page is highlighted) |
| Body Text | Dark gray/charcoal |
| Badge/Tag Background | Light gray with rounded pill shape |
| Card Borders | Thin, light gray |
| Footer Background | Slight gradient, lighter than body |

### Typography
- **Font Family:** Clean, modern sans-serif (likely system or web font like Inter/Segoe UI)
- **Headings:** Bold, dark/black, large sizing
- **Body Text:** Regular weight, dark gray, well-spaced
- **Navigation:** Medium weight, blue for links

### Layout Structure
- **Overall:** Central content column on a gradient background
- **Cards:** White with rounded corners, thin light-gray borders, subtle drop shadows
- **Grid:** Services and Tutoring pages use 3-column card grids
- **Homepage:** Two-column hero (main intro + Quick Links sidebar), then 3-column card row
- **Research & Ministry:** Two-column split layout
- **Books:** Horizontal 3-column card layout
- **Theme pages (Academic, Engineering, Creative):** Minimal single-column layout with reduced navigation

### Interaction Design
- **Buttons:** Bordered/outlined style (not filled), rounded
- **CTA Buttons:** "Explore", "Listen", "Visit Store" — outlined style
- **Platform Buttons (Music):** Compact, bordered grid of platform links
- **Tags/Badges:** Pill-shaped light gray badges for categorization (e.g., "Metamaterials", "Antennas")
- **Quick Links Card:** Highlighted with a slightly different (light blue) background color

### Responsive Notes
- Navigation wraps to multiple lines on different screen widths
- Cards and grids appear to be flex-based layouts
